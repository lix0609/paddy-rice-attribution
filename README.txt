===============================================================================
 Climate-enabled expansion and technology-led intensification of paddy rice
 in Jilin, Northeast China: attribution and CMIP6 projection (1985-2020)

 Analysis and figure code (CODE ONLY - no data included)
===============================================================================

This repository contains the analysis and figure-generation code for the study
of 35 years of paddy-rice production growth in Jilin Province, Northeast China.
The workflow fuses a 30-m satellite paddy map, county rice statistics and
gridded climate within a single reproducible framework:

  - additive LMDI decomposition of production growth (area vs. yield effects);
  - county fixed-effects panel regressions (area and yield vs. climate);
  - a Lobell-type climate/technology attribution with a bounded-range estimate;
  - interpretable machine learning (random forest + SHAP + partial dependence);
  - a delta-change CMIP6 projection (ensemble mean and nine individual models)
    with inter-model and response-parameter (bootstrap) uncertainty.

>>> IMPORTANT: This repository ships CODE ONLY. No data are distributed here -
>>> neither the raw inputs nor our derived/intermediate tables nor our
>>> station-level records. See Section 4 for what the code needs and where the
>>> public inputs can be obtained.

 2. REQUIREMENTS
-------------------------------------------------------------------------------

  Python >= 3.9 with the packages in requirements.txt:
     pip install -r requirements.txt

  Core libraries: numpy, pandas, scipy, matplotlib, scikit-learn, shap,
  statsmodels, linearmodels, rasterio, pymannkendall, requests
  (Pillow is optional, used only for 600-dpi TIFF export of figures.)

-------------------------------------------------------------------------------
 3. FILE PATHS (edit before running)
-------------------------------------------------------------------------------

  Each script defines absolute paths near the top (ROOT / PROJ / MID / PUB).
  Edit these to point to your local checkout and to the local folders where you
  place the inputs and want the outputs, for example:

     ROOT = "/your/path/jilin-paddy-rice-attribution"
     RAW  = "/your/path/raw"       # raw rasters and digitized statistics (you provide)
     MID  = "/your/path/data"      # intermediate tables (generated by the pipeline)
     PUB  = "/your/path/figures"   # output figures (create this folder)

-------------------------------------------------------------------------------
 4. DATA - NOT PROVIDED IN THIS REPOSITORY
-------------------------------------------------------------------------------

  We do NOT release any data with this code. To run the pipeline you must obtain
  the public inputs yourself and regenerate all intermediate tables.

  (a) RAW INPUTS - public (download from the original providers):

      - 30-m annual paddy-rice map of Northeast China (1985-2020):
          Zhang et al., 2025, Earth Syst. Sci. Data 17, 6851-6869.
          https://doi.org/10.5194/essd-17-6851-2025
      - 1-km monthly temperature & precipitation for China (1901-2017):
          Peng et al., 2019, Earth Syst. Sci. Data 11, 1931-1946.
          https://doi.org/10.5194/essd-11-1931-2019  (National Tibetan Plateau Data Center)
      - 1-km monthly CMIP6 historical & future climate for China:
          Hu et al., 2025, Sci. Data 12, 436. https://doi.org/10.1038/s41597-025-04761-y
          Science Data Bank: https://doi.org/10.57760/sciencedb.13546
      - ASTER Global DEM v3:
          NASA/METI, 2019. https://doi.org/10.5067/ASTER/ASTGTM.003

  (b) RAW INPUTS - restricted / our own (NOT redistributed):

      - County rice sown area and production: Jilin Statistical Yearbook
        (Jilin Provincial Bureau of Statistics, 2021), digitized by the authors.
      - Station-level rice phenology and agrometeorological yield records:
        the authors' own data; available from the corresponding author on
        reasonable request. (These are NOT provided in this repository.)

  (c) INTERMEDIATE / DERIVED TABLES (generated by the pipeline; NOT provided):

      The figure and analysis scripts read tables that are produced by running
      Stages A-D (Section 5) on the raw inputs above. You must generate these
      yourself; they are not included. The filenames the scripts expect include:

        county_year_panel.csv               county-year panel (area, yield, climate)
        climate_features_county_year.csv    agroclimatic features per county-year
        paddy_area_by_county_1985_2020.csv  satellite paddy area per county-year
        paddy_area_province.csv             provincial paddy-area series
        yield_panel_province.csv            provincial area / production / yield
        centroid_migration.csv              paddy centroid latitude / elevation
        climate_province.csv, county_static.csv, county_change_summary.csv
        county_at10_gradient.csv, province_at10_trend.csv
        temp_monthly_detail.csv, prec_monthly_detail.csv
        cmip6_county_monthly.csv, cmip6_county_monthly_allwin.csv,
        cmip6_gcm_monthly.csv               extracted CMIP6 climate per county / GCM
        projection_county*.csv, projection_bands.csv, projection_trajectory.json,
        projection_results.json, projection_summary.json, projection_upgrade.json,
        projection_uncertainty.json         projection outputs and uncertainty
        attribution_results.json, headline_numbers.json, mk_trends.json,
        regression_results.json, pdp_data.npz
        phenology_station_year.csv, phenology_matched_climate.csv,
        phenology_accumulated_temp.csv, phenology_temp_sensitivity.json  (from station data)
        issue1_attribution_range.json, issue3_response_forms.json  (revision analyses)

  In short: obtain the public rasters/statistics, run Stages A-D to rebuild the
  intermediate tables, then run the figure scripts (Stage E).

-------------------------------------------------------------------------------
 5. WORKFLOW (recommended execution order)
-------------------------------------------------------------------------------

  Stage A - Data preparation and feature engineering
     01_paddy_area_by_county.py     per-county annual paddy area from the 30-m map
     02_extract_resumable.py        resumable variant of the extraction
     08_climate_features.py         county agroclimatic features (GS mean T, GDD,
                                     AAT>=10C, warmest-month T, May T, precip, ...)
     12_phenology_temp_sensitivity.py  station phenology / thermal accumulation (SI)

  Stage B - Trends, decomposition and panel models
     03_analysis.py                 MK / Theil-Sen trends; additive LMDI
                                     decomposition; headline numbers
     05_panel_regression.py         county fixed-effects panel regressions

  Stage C - Climate/technology attribution and interpretable ML
     09_attribution_ml.py           county detrending; random forest; grouped CV;
                                     SHAP; partial dependence; bounded-range split

  Stage D - CMIP6 projection
     download_cmip6.py              download CMIP6 raster inputs
     19_cmip6_extract.py            ensemble-mean CMIP6 features per county
     27_cmip6_extract_allwindows.py all three future windows (2030/2055/2085)
     30_cmip6_extract_gcm.py        per-GCM extraction (nine models)
     11_delta_projection.py         delta-change projection of climate-driven yield
     20_cmip6_project.py            projection to production changes
     25_projection_upgrade.py       projection refinement
     28_projection_trajectory.py    temporal trajectory (windows)
     31_uncertainty.py              inter-model spread + response-parameter
                                     (four functional forms, 1000-sample bootstrap)

  Stage E - Figures (final / published versions)
     22_fig_growth_lmdi.py          Fig 2  (production growth + LMDI)
     32_fig45_revise.py             Fig 4  (attribution) and Fig 5 (T-yield response)
     33_fig7_maps.py                Fig 7  (county projection maps)
     34_fig8_panels.py              Fig 8  (projection summary panels)
     17_figS_thermal.py             Fig S4 (thermal constraint and phenology)
     35_figS6_revision.py           Fig S6 (projection-robustness support)
     23_graphical_abstract.py       graphical abstract
     06_maps.py / 07_figures_pub.py earlier consolidated figures (Fig 1, 3, 6, SI)

-------------------------------------------------------------------------------
 7. NOTE ON SCRIPT NUMBERING
-------------------------------------------------------------------------------

  The numeric prefixes (01_, 02_, ... 35_) reflect the DEVELOPMENT and execution
  history, not a single clean pipeline. Several figures went through successive
  revisions (e.g. 04/13/14/16/24/26/29 for projection and attribution figures);
  these earlier versions are retained for provenance. The FINAL, published
  figures are produced by the scripts listed in Sections 5E and 6.
